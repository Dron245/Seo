Расширенный набор функций

Мы способны реализовать абсолютно любой функционал, любые идеи, которые необходимы для работы как маленького, так и большого интернет-магазина

Корзина

Неотъемлемая часть интернет-магазина. Покупатель сможет сам оформить заказ прямо на сайте. Настраиваемые поля позволяют убрать все лишнее и сделать покупку максимально комфортной.

Фильтр товаров

Работает на страницах категорий (подкатегорий). Можно задать любые произвольные параметры для фильтрации товаров (для каждой категории свои).

Экспорт / Импорт

Модуль позволяет загружать/выгружать товары из прайс-листов в формате EXCEL. С помощью модуля можно обновлять цены, количество и другую информацию для уже загруженных товаров.

Онлайн оплата

Интеграция с платёжной системой. Возможность принимать на сайте оплату с банковских карт, через Яндекс Деньги, Webmoney и другими способами.

Блок доставки

Функция для создания неограниченного количества любых способов доставки с разной стоимостью для каждого варианта.

Каталог / Меню

Многоуровневое многофункциональное горизонтальное меню с возможностью создания сложной, но удобной навигации по интернет магазину.

Поиск

Удобный поиск товаров по сайту позволит покупателю быстро найти нужный товар в каталоге. Работает с автодополнением (система автоматически покажет совпадения).

Личный кабинет

Личный кабинет с удобным интерфейсом, который позволяет хранить информацию, к которой пользователь может вернуться в будущем. Это облегчает совершать покупки.

Уведомления

Автоматические уведомления администратора и покупателя по электронной почте после оформления заказа, при смене статуса заказа и регистрации на сайте.
